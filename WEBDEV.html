Web Development

Web development, also known as web programming, refers to the process involved in building and maintaining websites.
It describes the operations taken to make a website appealing and functional, as well as to provide its users with a seamless experience.
Web development is different from web design. It majorly involves the coding and programming operations that ensures a website function properly.
Web development results in the creation of anything from a single plaintext webpage to a complex social network or web application.

Web development is usually executed in three major ways:

- Front-end development.
- Back-end development.
- Database technology.

Front-End Development:

also known as client-side coding, front-end development deals with the development of the layout, interactivity, fonts, menus, design, and other parts of a website that the end user sees when on the website.
It also enables users to interact with the website, highlight text, expand or reduce images, watch videos, etc.
The coding languages used in front-end development enables the website to operate without having to constantly communicate with the internet.
Front-end development is usually carried out with three codes: CSS, HTML, and JavaScript.

HTML - HTML is the abbreviated form of Hyper Text Markup Language.
It is a special code that is used to "mark up" text in a bid to turn it into a webpage.

CSS - CSS is the short form of Cascading Style Sheets, and it is a code used for defining the style rules that determine how the webpage appears.

JavaScript - JavaScript is a coding language that is popularly used to integrate functionality and interactivity in webpages.

Back-End Development:

also known as server-side coding, back-end development deals with everything that happens “under the hood” of a website.
That is to say, it deals with everything that makes websites functional but which users cannot see.
The back-end is where website data is stored, without which the front-end would not exist.
It also includes the server hosting the website, the application that runs it, and a database that holds the data.
Server-side coding employs computer programs to ensure that the aforementioned server, application, and database work together in a seamless manner.
Back-end development is usually done on a number of coding languages, such as PHP, Ruby for Rails, Node.js, Python, etc.
